📌 Project Overview

fake_json_data is a data generation project designed to create realistic, structured JSON datasets for use in:

Frontend development

Backend API testing

Database seeding

Machine learning experiments

E-commerce mock systems

Chatbot product simulation

The project generates clean, consistent, and production-style JSON records such as products, users, carts, or orders.

🎯 Purpose

During development, real production data is often unavailable. This project solves that problem by generating:

Well-structured JSON

Realistic product names

Proper pricing formats

Nested rating objects

Image URLs

Categories and metadata

Perfect for:

React / Next.js frontend testing

FastAPI / Django backend testing

Postman API testing

ML training mock datasets
